# mcp-taskwarrior

Author: Bruce Kroeze

A simple MCP Server that facilitates management of tasks in Taskwarrior.

Provides 4 tools:

- add_task
- update_task
- delete_task
- list_tasks

## Usage

```bash
pnpm install
```

In Cursor, Goose, Claude or your favorite MCP client, choose "Command" style MCP, then `/path/to/this/repo/src/tasks.ts` as the argument.
